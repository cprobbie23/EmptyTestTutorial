1. As a user, when navigating to the home page, I want to see a list of users
	--Acceptace criteria--
	* I am on the main page
	* I can see a table of users
	* The table is populated
	* The table has values for all columns
	* The table is ordered by creation date, most recent first
	* The table shows the 20 users maximum

2. As a user, when looking at the users table, I want to save a user
	--Acceptace criteria--
	* I can enter a first name
	* I can enter a last name
	* I can enter an address
	* I can save a user
	* On successful save, I get redirected to the main page
	* On redirect, I can see my new user in the list
